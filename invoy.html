Client Experience Internship

Overall Objective

During this past summer, I received the opportunity to be a Client Experience Intern at Invoy Technologies in Aliso Viejo, CA. During this time I focused on enhancing client communication and brainstorming new ways to improve the overall user experience of the application. Along with this, I tested the company's app daily for any potential software bugs and took part in weekly software meetings in order to come up with new design features for the application.

Task Completed

By having this internship I was able to do the following:

  • Collaborate in weekly software design review meetings to produce updates for the existing app
  • Acceptance testing of the companies app and internal website to detect any bugs and create new ideas that will improve the overall user experience
  • Manage the Client Experience team's Trello board to improve communication and collaboration with the software teams
  • Manage and distribute incoming client request tickets via Zendesk to ensure timely delivery for end users

Design Results

The app has been succesfully updated to version 3.5.767. As a team we were able to implement a learning feature and a chat feature for users to engage with.
